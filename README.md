**Customer Churn Analysis – Power BI Dashboard**
This project analyzes a telecom customer churn dataset to identify key factors influencing churn and to help businesses improve customer retention.

**Key Features**
- KPIs: Total Customers, Churned Customers, Churn Rate
- Churn Breakdown: Churn by Gender, Churn by Contract Types
- Customer Insights: Customer distribution by Payment Method
- Visual Storytelling: Clean UI with professional layout and shadows
- Actionable Insights: Identified high-risk segments based on contract and payment types

**Dashboard Pages**
**Page 1 – Customer & Churn Insights Overview**
- KPI Cards
- Churn by Contract
- Churn by Gender
- Customer Count by Payment Method

**Page 2 – Customer Profile Breakdown**
- Senior Citizen Distribution
- Age Groups
- Tenure Groups
- Payment Methods
- Contract Types
- Monthly Charges (Binned)
